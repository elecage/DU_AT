# 장애물 탐지 바닥센서

## 개요
  * **장애물 탐지 바닥센서**는 휠체어를 사용하는 장애인들의 후진 시 충돌 위험으로부터 보호하기 위해 휠체어에 장착하여 후방의 장애물의 존재 여부를 알려주고 후방 영상을 전시하는 기능을 갖는 보조기기이다.
